# 🏛️ Painel de Aprovação SEDES-DF

> Aplicativo de estudos para gerenciamento de aprovações - SEDES-DF

## 📋 Sobre o Projeto

Painel de Aprovação SEDES-DF é uma aplicação web desenvolvida em HTML com instalador para Windows, focada em facilitar o processo de aprovação de documentos e solicitações.

## ✨ Funcionalidades

- Interface intuitiva para aprovação de documentos
- Instalador automático para Windows
- Aplicação responsiva baseada em HTML

## 🚀 Início Rápido

### Pré-requisitos
- Windows 7 ou superior
- Navegador moderno

### Instalação
1. Baixe o instalador Windows
2. Execute o arquivo de instalação
3. Siga o assistente de instalação
4. Acesse a aplicação após a instalação
